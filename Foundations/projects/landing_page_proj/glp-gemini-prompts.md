## TIPS FOR USING THESE PROMPTS:

1. **Add style modifiers:** Add terms like "hyperrealistic," "professional photography," "8K," "studio lighting" for higher quality

2. **Specify dimensions:** Always include desired dimensions like "1200x800px" or "square format for Instagram"

3. **Include negative prompts:** Add what you don't want, like "no text," "no watermarks," "no blur"

4. **Iterate:** Start with the base prompt and refine based on initial results

5. **Consistency:** Use similar color schemes and styles across all images for brand cohesion

Example enhanced prompt:
```
[Base prompt] + "hyperrealistic, professional product photography, 8K resolution,
studio lighting, no text overlays, no watermarks, clean minimalist style,
trending on Behance, award-winning design"
```
